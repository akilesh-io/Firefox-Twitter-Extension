# Bring Back Blue Bird

Extension to remove the current **X** logo and bring back the classic blue bird on Twitter/X.

## Firefox Add-on

👉 [Install from Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/twitter-blue-bird/)

## Features

- Replaces the **X** logo with the classic blue bird
- Works across Twitter/X pages
- Firefox support

---

## Installation

1. Open the Firefox Add-ons page
2. Click **Add to Firefox**
3. Visit Twitter/X and enjoy the blue bird again
